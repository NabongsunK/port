# 1. 개요

## 1.1 프로젝트 요약

### 1. 일정

2024년 3월 27일 ~ 2024년 4월 2일(6일)

### 2. 프로젝트 요약

- nextJS와 typescript를 이용한 반응형 웹페이지 구현
- vercel을 이용한 홈페이지 배포를 함
- 포토폴리오 용 웹사이트 제작

### 3. 나의 역할

- nextJS로 홈페이지 주요기능 구현

### 4. 느낀점/배운점

- nextJS, typescript 문법에 익숙해짐
- tailwind로 반응형 웹페이지 구현에 익숙해짐

### 5. 개발시 발생한 문제점/해결책

- MixItup 효과를 구현하려 했을때, 자료가 얼마 없었으나, 해외 자료를 참고해 문서작성을 함

![](https://velog.velcdn.com/images/nabongsun/post/dae590c5-cb33-423e-a090-39046f793e7b/image.png)

# 2. 사용된 기술

- contentlayer: nextJS에서 md, mdx파일을 html형식으로 바꿔주는 모듈, 블로그 구현에 사용됨
- react-mixitup: react에서 MixItUp효과를 구현하는것을 도와주는 모듈
- swr: react에서 서버데이터를 관리하는 모듈 다크모드를 위한 변수 상태관리에 사용됨
- tailwind: css 프레임워크, 반응형 웹디자인을 위해 사용됨
- typescript: 정적인 데이터의 자료형을 선언해 생산성을 높이기 위해 사용됨
