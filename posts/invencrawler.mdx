## 2.2 KoNlPyf를 이용한 한국어 데이터 전처리

```
if ':'in date:
  date = today
date = date[0:3]+str(((int(date[3:5])-1)//5)*5+1)
nounss = okt.nouns(title)
tmp = []
for noun in nounss:
  if(len(noun)>1):
    tmp.append(noun)
if date in db.keys():
  db[date]+=tmp
else:
  db[date] = tmp
```

date의 종류가 두가지 있는데, 48시간 이내 글에는 12:34 이렇게 시간으로 표시되고, 그 이후 글에는 03.12 이렇게 날짜로 표시된다. 3일마다 끊어서 db Object에 저장
